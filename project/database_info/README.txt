In this folder there are XML files and php scripts that created our database
it is in the sql zip file named kanji.sql(2).
JMdict_e is the english version of the japanese word database created by
Jim Breen.
kanjidic2 is the database that holds information about kanji also done
by Jim Breen.
His homepage with links to these projects is http://www.csse.monash.edu.au/~jwb/

these scripts shouldn't be run because they might alter the information in the
database.

used various sites for reference:
http://www.w3schools.com/ - for php, js, jquery, sql, css, html, bootstrap help
php.net/ - for help using some php apis
api.jquery.com - for jquery api help
http://getbootstrap.com/ - for help using the bootstrap features and downloading css

If the Database doesn't install in a readable format or corrupted format, then use 
http://www.toptal.com/php/a-utf-8-primer-for-php-and-mysql to make sure that
your mysqli and php are in the right form to take in utf8 encoded data.
